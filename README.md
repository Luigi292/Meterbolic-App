
# AWS Lex Chatbot Starter

This is a ready-to-deploy chatbot frontend using Next.js, TypeScript, and AWS Lex.

## Setup

1. Copy `.env.example` to `.env.local` and fill in your credentials.
2. Run:

```bash
npm install
npm run dev
```

## Deploying with Amplify

```bash
amplify init
amplify add hosting
amplify publish
```
